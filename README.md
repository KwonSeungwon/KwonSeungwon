### 🚀 About Me
안녕하세요! 저는 **Java/Kotlin** 기반의 프로젝트를 주로 개발합니다.

최근에는 **AI(Claude, Cursor, Copilot)**를 적극적으로 워크플로우에 녹여내어, 단순 반복 코딩보다는 **비즈니스 로직 설계와 아키텍처 고도화**에 집중하는 **'바이브 코딩'** 스타일의 개발을 지향하고 있습니다.

- 🛠 **Main Stack:** Java 21, Kotlin, Spring Boot, K8s

- 🌐 **Interest:** Distributed Systems, Cloud Native, Prompt Engineering
